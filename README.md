## 使用方法(Usage)
因台灣證交所資料更新時間限制，在terminal 執行本程式即可自動儲存昨日的股票交易資訊。

第一版以json檔與csv檔做儲存。並存在與本程式同路徑中。
同時可以用csv檔讀取每天儲存的股票交易資訊擷取單一股票資料做表格呈現

下一版計畫更改成以.db檔(SQL) 操作。
